# Ejemplo de partida bloque 4

Este ejemplo está basado y combina varios ejemplos de Espressiff, junto con código proprio y blibliotecas de terceros. 

Implementa un servidor TCP con un protocolo "binario" sencillo (tramas de tamaño constante). De momento permite modificar el estado del LED rojo 
desde un cliente de interfaz gráfico implementado con Qt.

Tambien implementa una consola de comandos de terminal y una tarea que gestiona un reloj por el LCD.
